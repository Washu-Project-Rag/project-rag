# Retrieval-Augmented Generation (RAG) for Corporate Wikis/Documentation


The goal of this project is to build an AI-powered chatbot LLM-based chatbot that provides 
accurate answers to user queries related to Engineering subjects using Retrieval-Augmented 
Generation (RAG). RAG allows us to incorporate information retrieval before response generation, 
thereby creating contextually supported answers backed by real, traceable sources. By utilizing RAG, 
we can reduce AI hallucinations and the need to retrain LLMs with new data. 

### Methods

**Dataset:** an English Wikipedia dataset dump snapshot (e.g., “pages-articles”).

The data is pre-processed using the RegEx Python package, filtered by specific keywords related 
to the "Engineering" academic subject field to reduce source data size and memory cost.
